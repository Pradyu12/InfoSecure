# InfoSecure React Starter

This repository contains a React.js web application with proper component architecture.

## Project Structure

- `src/App.jsx` - Main application component
- `src/main.jsx` - React entry point  
- `src/index.css` - Global styles
- `src/components/sections/` - Page components
- `src/components/ui/` - UI components (shared)
- `src/hooks/` - Custom React hooks
- `src/utils/` - Utility functions

## Key Features

✅ **Modular Architecture** - Components are separated and focused
✅ **Reusable Hooks** - Custom hooks for animations and state management
✅ **Data Abstraction** - Constants centralized in utils/data.js
✅ **Clean Architecture** - Follows React best practices
✅ **Scalable Structure** - Easy to add new components as needed

## Current Components

- **App.jsx** - Main application component
- **Hero.jsx** - Hero section with stats counters and animations
- **About.jsx** - About section
