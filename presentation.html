Dynamics

Interactive Experience

Scroll-Triggered Reveals

Every section fades + slides up as it enters the viewport. Staggered delays create a wave effect across card grids.

const observer = new IntersectionObserver(
  entries => entries.forEach(e =>
    e.target.classList.toggle('visible', e.isIntersecting)
  ), { threshold: 0.15 }
)

Continuous Animations

  • 3D Wireframe Torus — canvas-based rotating torus in hero, depth-sorted vertices with glow
  • Typewriter — cycles 4 phrases every 2.8s with cursor blink
  • Counters — numbers animate from 0 to target over 1.5s
  • Marquee — client names scroll horizontally, pause on hover
  • Tilt — cards rotate up to 6° following the mouse
  • Progressive disclosure — toggle extra solution details inline
Backend-Free Contact

Form → GitHub Issue

No server, no database. The form posts directly to the GitHub Issues API using a fine-grained PAT.

How It Works

POST https://api.github.com/repos/{repo}/issues
Authorization: Bearer {PAT}
{
  title: "Contact: John <j@a.com>",
  body: "**Name:** ...\n**Message:** ...",
  labels: ["contact"]
}

Deployment Flow

  • PAT stored as GitHub Actions secret CONTACT_PAT
  • Workflow replaces __GITHUB_PAT__ placeholder via sed at deploy time
  • Built files published to gh-pages branch
  • No backend = zero server costs, zero maintenance
